# Basic chat app

Basic chat app is an application for chatting in realtime, to implement socket.io and socket.io-client with react

## Feature public chat

- Join room
- Sendto by id
- leave room

## Feature admin chat

- Join room
- Sendto by username
- leave room

## Account to login as admin

format: (username, password)

- ('admin','admin')
- ('testAd','admin123')
- ('test123a','test123')

## Usage

- Install `node.js`
- Run `npm install` inside backend and frontend folder
- Open [this link](http://localhost:3000)

## Technologies

- React Typescript
- Socket
- Tailwind
